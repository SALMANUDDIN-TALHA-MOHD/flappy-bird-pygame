# Code of Conduct

## Our Pledge

We pledge to make participation in this project a harassment-free experience for everyone, regardless of experience level, education, or 
background.

## Our Standards

**Positive behaviors:**
- Being respectful and inclusive
- Welcoming questions from learners
- Providing constructive feedback
- Being patient with beginners
- Focusing on learning and growth

**Unacceptable behaviors:**
- Harassment or discriminatory language
- Trolling or insulting comments
- Publishing private information
- Unprofessional conduct

## Educational Context

This is a student learning project. We welcome:
- Questions about game development
- Bug reports with reproduction steps
- Constructive code feedback
- Learning-focused discussions

## Enforcement

Report issues to the project team. All complaints will be reviewed promptly.

## Our Commitment

As students, we commit to:
- Creating a welcoming learning environment
- Responding respectfully to questions
- Crediting sources appropriately
- Supporting the open-source community
